はじめる前に

初回起動時の警告について

Apple認証を受けていないアプリのため、初回にこの警告が出ます。

  1. 「完了」をクリック（ゴミ箱に入れない）
  2. システム設定 → プライバシーとセキュリティ を開く
  3. 「このまま開く」をクリック

※ 初回のみ。2回目以降は通常通り起動できます。

Claude APIキーの取得

AI（Claude）を利用するため、APIキーとクレジットの購入が必要です。
従量課金制で、1回の質問あたり約1〜3円。毎日100回使っても1日100〜300円程度です。

  1. Anthropic Console でアカウント作成
  2. Settings → Billing でクレジット購入（$5〜）
  3. Settings → API Keys でキー発行
  4. アプリにキーを入力

キーはMacのアプリ内にのみ保存され、外部に送信されません。
